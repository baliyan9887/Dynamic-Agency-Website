3. Sign up

First of all we need to check our form fields are empty or not. if our form field is empty then push an error to user:

             

 $errors = array();             
             
 if (isset($_POST['reg'])) {

4. Access the Dashboard

Congratulations ! we succesfully incorporate user's informatin into our database. Now we need to forward the user to the dashboard. For those pages that can only be accessed by the registered members, we need to put an the top of the dashboard page. Notice that we're setting an authorized session variable in the dashboard code above. On the top of pages we need to protect, we check for that session variable. if user is authorized, we show him the protected content, otherwise we direct him to login form.

Here is the code to your protected pages: 

             

 <?php include('sec-login-server.php');
        //if user not logged in, they cannot access this page
