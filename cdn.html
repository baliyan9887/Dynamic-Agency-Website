1. Prettify.js

An embeddable script that makes source-code snippets in HTML prettier.

  • Works on HTML pages.
  • Works even if code contains embedded links, line numbers, etc.
  • Simple API: include some JS & CSS and add an onload handler.
  • Lightweights: small download and does not block page from loading while running.
  • Customizable styles via CSS. See the theme gallery.
  • Supports all C-like, Bash-like, and XML-like languages. No need to specify the language.
  • Extensible language handlers for other languages. You can specify the language.
    •                 
     <script src="https://cdn.jsdelivr.net/gh/google/code-prettify@master/loader/run_prettify.js"></script>

    2. Polymer.js

    Polymer.js is an open source JavaScript library developed by Google. It is developed to allow reusing the HTML elements for building applications with components.

  • Provides the simplest way to create custom HTML elements because its library is built on top of web standards API.
  • Uses Google material design to develop hybrid mobile applications. So, it is fast and easy to develop mobile applications.
  • Facilitates you to create your own customized and reusable elements by using polyfills web components specification.
  • Supports one way and two way data binding.
  • Provides polymer command line interface to manage complex projects.
  • Fast and easy hybrid mobile application development.
    •                 
     <script src="https://cdnjs.cloudflare.com/ajax/libs/polymer/0.5.6/polymer.js"></script>                 
 <script src="https://cdnjs.cloudflare.com/ajax/libs/polymer/0.5.6/polymer.min.js"></script>

    3. Smooth-scroll.js

    A lightweight script to animate scrolling to anchor links. Smooth Scroll works great with Gumshoe.

  • Works on HTML pages.
  • Include Smooth Scroll on your site.
  • Add the markup to your HTML.
  • Initialize Smooth Scroll.
    •                
     <script src="https://cdn.jsdelivr.net/gh/cferdinandi/smooth-scroll@15.0.0/dist/smooth-scroll.polyfills.min.js"></script>

    4. Gumshoe.js

    A simple vanilla JS scrollspy script. Gumshoe works great with Smooth Scroll.

  • Works on HTML pages.
  • Include Gumshoe on your site.
  • Add the markup to your HTML.
  • Initialize Gumshoe.
  • Add Styling.
    •                
      <script src="https://cdn.jsdelivr.net/gh/cferdinandi/gumshoe@4.0.0/dist/gumshoe.polyfills.min.js"></script>

    Ember.js is a JavaScript framework for creating ambitious web applications.

  • Works on HTML pages.
  • Write less code with templates that automatically update.
  • Retrieve models, with rich relationships, from your server.
  • Conventional app structure helps your app grow without messy code.
